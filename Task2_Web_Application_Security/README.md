# Web Application Security Analysis using OWASP ZAP

This project was completed as part of my **Cyber Security Internship**.

## Objective
The objective of this task is to analyze a vulnerable web application and identify common web security vulnerabilities.

## Tools Used
- Kali Linux
- OWASP ZAP
- WebGoat

## Process
1. Installed and ran the WebGoat vulnerable web application.
2. Used OWASP ZAP to perform an automated vulnerability scan.
3. Analyzed detected security issues in the application.

## Vulnerabilities Identified
- SQL Injection
- Absence of Anti-CSRF Tokens
- Security Header Misconfiguration
- Session Management Issues

## Output
The OWASP ZAP scan generated a vulnerability report highlighting potential security risks in the application.
